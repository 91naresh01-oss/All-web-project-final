# 📐 Task Format

## Template
```markdown
## ✅ Task Completed Summary

| Item | Detail |
|------|--------|
| **Task** | What was done (1 line) |
| **Files Changed** | X files |
| **Files Created** | X files |
| **Status** | ✅ Complete |
```

## Rules
- ગુજરાતી only
- Tables use — no long sentences
- Small task (1-2 files) → short table
- Large task (6+ files) → full table + Changes Made table

## Changes Made Table (optional for larger tasks)
```markdown
| # | File | Change |
|:-:|------|--------|
| 1 | `file.ext` | Short description |
```
