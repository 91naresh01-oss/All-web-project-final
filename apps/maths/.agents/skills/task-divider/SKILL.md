# ✂️ Task Divider

## Rules
1. **સ્પષ્ટ કામ** → Plan વગર સીધું execute
2. **મોટું કામ** → Logical parts માં divide, દરેક part માટે અટકવું નહીં
3. **One part at a time** → Part-1 complete + verify → Part-2
4. **Token limit** → 50+ items = 30-40 batch
5. **Final** → summary format see `task-format` workflow
