# English Words Learn - Gyan Academy

## Project Overview
Learn English words with meaning, pronunciation, verb forms, synonyms, antonyms, and examples in multiple Indian languages (Gujarati, Hindi, etc.).

## Features
- **Spelling Journey**: Learn words level by level.
- **Grammar Guide**: Grammar rules and practice tests.
- **Progress Sync**: Login with Google to save progress to the cloud (Firebase).
- **Offline Support**: PWA enabled.

## Tech Stack
- Next.js 14
- Tailwind CSS
- Firebase (Auth & Firestore)
- Radix UI / Shadcn UI
- Lucide React Icons

## Deployment
Deployed on Vercel with GitHub auto-deploy.
